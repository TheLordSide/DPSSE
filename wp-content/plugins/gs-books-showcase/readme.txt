== Description ==

Best Responsive Book Showcase plugin for WordPress to display Book details like Cover, Author, Published On, Publisher, Reviews, Rating, Translator, ISBN, ASIN, Total Pages / Length, Country, Language, Book Dimensions, File Size (if e-book), Author Biography & many more. Display anywhere at your site using shortcodes like [gs_book_showcase theme="gs_book_theme1"] & widgets. GS Bookshowcase plugin packed with tons of controlling options & different themes to showcase books elegantly with eye-catching effects.

GS Books Showcase plugin is simple, flexible & powerful.

### GS Books Showcase's shortcode usage

Basic Usage - 
`
[gs_book_showcase theme="gs_book_theme1"]
`

**GS Books Showcase's Shortcode attributes Usage**
`
[gs_book_showcase num="10" theme="gs_book_theme1" cols="4" group="cookbook,fiction" cats_name="none" desc_limit="100" order="DESC" orderby="date"]
`

**Shortcode PHP Usage**
`
<?php echo do_shortcode( '[gs_book_showcase theme="gs_book_theme1"]' ); ?>
`

== Installation ==

Install GS Books Showcase plugin as a regular WordPress plugin. Here is different ways to install GS Books Showcase plugin :

*   [Visual Installation & Usage Guide &raquo;](https://docs.gsplugins.com/gs-books-showcase)

= Install by Search Plugin =
- In your Admin, go to menu Plugins > Add New
- Search GS Bookshowcase
- Click to install
- Activate the plugin
- GS Bookshowcase post type will show below Posts / Pages

= Install by Upload Plugin =
- Download the latest version of GS Bookshowcase (.zip file)
- In your Admin, go to menu Plugins > Add New
- Select **Upload Plugin**
- Click on **Choose File**
- Select downloaded gs-bookshowcase-pro.zip & click on **Install Now** button
- Activate the plugin
- GS Bookshowcase post type will show below Posts / Pages

= Install Plugin using FTP =
- Upload gs-bookshowcase-pro folder inside the /wp-content/plugins/ directory
- Go to WordPress dashboard > Plugins > Activate the GS Bookshowcase plugin
- GS Bookshowcase post type will show below Posts / Pages

* Put [gs_book_showcase theme="gs_book_theme1"] shortcode or select another theme, where you need to show the Books.

== Frequently Asked Questions ==

= How do I add a new Book? =
GS Book Showcase post type will show below Posts / Pages. To add new Book, go to GS Books Showcase > Add New Book, then add Title, Desc, Cover, Author, Published On, Publisher, Reviews, Rating, Translator, ISBN, ASIN, Total Pages / Length, Country, Language, Book Dimensions, File Size (if e-book), Author Biography. Now you are ready to Publish.

= What will be Book image dimension? =
Recommended image size 600px X 900px for perfect view on various devices.

= How to use shortcode inside page templates? =
WordPress has a great function, do_shortcode(), that will allow you to use shortcodes inside your theme files. For example, to output Books Showcase in a Theme file, you would do this: 
`
<?php echo do_shortcode('[gs_book_showcase theme="gs_book_theme1"]'); ?>
`

= How can I add GS WordPress Books Showcase plugin in my WordPress site? =
Install GS Books Showcase plugin as a regular WordPress plugin. Here is different ways to install GS Books Showcase plugin :

- [Visual Installation & Usage Guide &raquo;](https://docs.gsplugins.com/gs-books-showcase)
- Put [gs_book_showcase theme="gs_book_theme1"] shortcode or select another theme, where you need to show the Books.

= Is there any limitation of adding books at GS Book Showcase plugin for WordPress? = 
There is no limitation to add a number of books, so you can add as many books as you want.

= How can I use shortcodes to display books in my online book store? =
First install the GS Books Showcase plugin in your WordPress website. Then add the shortcode [gs_book_showcase theme="gs_book_theme1"] where you wish to display books.

= What is the recommended image size of a book for GS Books Showcase plugin? =
600px X 900px is the recommended image size for Books Showcase plugin. Other dimensions may not give you the proper outlook of your book design. 

= What are the design options available for GS Books Showcase plugin? =
